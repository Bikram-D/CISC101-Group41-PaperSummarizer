# Paper Summarizer
Created for the Paper Summarizer Project for the CISC 101 course.
## [System Prompt](https://github.com/Bikram-D/CISC101-Group41-PaperSummarizer/blob/main/system_prompt.md)
The system prompt describes how the paper summarizer should communicate with users. It includes the instructions for the LLM on what inputs are required, the boundaries for the paper summarizer, and the required outputs. 
## [Modules](https://github.com/Bikram-D/CISC101-Group41-PaperSummarizer/tree/main/modules)
The modules are the reference framework defining how each one works, what it requires for each, along with safety rules for each.
The modules for the Paper Summarizer are the following:
- Module 1: Intake & Setup
- Module 2: Section Loop
- Module 3: Guardrails
- Module 4: Rendering & Refinement
- Module 5: Citation Extractor
- Module 6: Key Contributions Summarizer
